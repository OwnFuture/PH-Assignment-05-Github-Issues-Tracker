1. What is the difference between var, let, and const?

Answer : 

var =>> A function-scoped variable declaration in JS (old method). It is hoisted and accessible throughout the whole fnction.

let =>> A block-scoped variable declaration introduced in ES6. It exists only inside the block where it is defined.

const =>> A block-scoped variable declaration whose value cannot be reassigned after initialization.

2.  What is the spread operator (...)?

Answer :

The spread operator expands elements of an array or object into individual elements.

3. What is the difference between map(), filter(), and forEach()?

Answer :

map() =>> Creates a new array by applying a function to each element.

filter() =>> Create a new array with elements that pass a condition.

forEach() =>> Executes a function for each element, but doesn't return a new array.

4. What is an arrow function?

Answer :

An arrow function is a shorter syntax for writing function in JavaScript. It uses the => (arrow) symbol.

5. What are template literals?

Answer :

Template literals are a JavaScript feature used to create strings easily using backticks . They allow embedding variables and expression.
